# How to run UT
* Create build directory under project root directory
```
$mkdir build
```
* Move to build directory, and run cmake
```
$cd build
```
```
$cmake ..
```
* Build and run tests
```
$make all
```
```
$make test
```
* Generate coverage report
```
$make coverage
```
